# Get-LANDeskColumnSets
## SYNOPSIS
The Get-LANDeskColumnSets function returns a list of available Column sets.

## SYNTAX

```
Get-LANDeskColumnSets
```

## DESCRIPTION
The Get-LANDeskColumnSets function returns a list of available Column sets.
This can be used to determine the columns available to each set.

## EXAMPLES

### -------------------------- EXAMPLE 1 --------------------------
```
Get-LANDeskColumnSets
```

Returns a list of available column sets.
